## ■ サービス概要

原神の編成紹介動画を検索・評価・おすすめ動画の共有が行えるサイト。任意の原神キャラクター1~4人を選択することで該当キャラクターの編成紹介動画（Youtube埋め込み）、キャラビルドなどの編成情報を参照できる。

ログインすることでおすすめ編成紹介動画を共有したり、編成評価したり、お気に入り登録することができる。

## ■ このサービスへの思い・作りたい理由

私自身が原神の編成紹介動画をYouTubeで投稿したり、まとめホームページを作成したりしてきた経験があります。
原神関連の動画投稿者、情報発信者も多く、情報が溢れており、プレイヤー目線で下記のような改善ができないかと考えていました。

- Youtube上での検索では、登録者数の多い投稿者の情報ばかりが候補として表示されやすいです。編成の深掘、丁寧な解説、最適アクション解説をしている動画は他にもたくさんあり届ける機会を作れないか。
- 大量の動画が存在するため、指定した4体の編成ピンポイントでの動画の検索が難しく、原神編成検索に特化したサービスを作れないか。
- 上記を実現できるサービスはあるが、動画投稿者の立場から「動画」を結びつけることで、よりプレイヤーが視覚的にわかりやすい情報を提供できないか。
- 発信者の一方的評価であることが多く、プレイヤー同士でも情報共有できるプラントフォームを作れないか。またその中で推しクリエーターの応援の機会などに繋げられないか。

このような改善に対するソリューションを実現するためにアプリを作成したいと思います。また、自分もこのアプリのユーザーとして、200本以上の自分の編成紹介動画もより届ける機会に繋げたいと思っています。

## ■ ユーザー層について

### 10代後半〜30代前半　原神プレイヤー

中でも、検索を行う対象は新規プレイヤーやライトプレイヤーをメインターゲット
おすすめ動画共有・評価を行う対象は長年プレイしており複数のキャラを所持しているプレイヤーをメインターゲットと設定します。

(理由：わたしのYoutubeアナリティクスの視聴者層が 「18-24歳:43.1% / 25-34歳:39.9%」と全体の8割以上を占めるため。)

## ■サービスの利用イメージ
「YouTubeの原神編成紹介」をメインにした情報共有・意見交換サイト(SNS)イメージになります。

### -検索について-

### おすすめ動画検索
原神のキャラクターを1〜4対選択し検索することで、該当キャラクターの含まれるおすすめ動画一覧が表示されます。
キャラクター4体を選んで検索した場合は、該当編成のおすすめ動画に合わせて、該当編成の平均評価も表示されます。

もし検索した編成の動画が投稿されていなければ「リクエストボタン」が表示され動画のリクエストをすることができます。

### 編成評価検索
原神のキャラクターを1〜4対選択し検索することで、該当キャラクターの編成評価一覧が評価の高い順から出現するようにします。

### -投稿について-

### おすすめ動画のシェア
ログイン者限定で、任意の1〜4対キャラを選択し「おすすめ動画」のYoutube埋込みを共有することができます。
その際に動画のおすすめポイントと各キャラに対して、動画で紹介されている武器や聖遺物（装備）などのビルド情報も登録することができます。（必須ではない）

### 編成評価
ログイン者限定で、任意の4対キャラを選択し「編成に関する評価(5段階)・コメント」を共有することができます。

### その他
・ログイン者限定で、編成検索した際に、該当の編成が出てこなければ紹介動画をリクエストをすることができます。
　通知設定をしておけばリクエストのお知らせ通知を受け取ることもできます。(動画投稿者などに使ってもらいたい機能です。)
・ログイン者限定で、気に入ったおすすめ動画をお気に入りリストに登録することができます。

## 価値

●プレイヤー向けの価値

「原神の編成」に特化した情報収集が行えるページになっており

- 自身が調べたい編成の視覚的に検索・比較することができます。
- 代替えの編成の発見や、新しいシナジー考案などに活用できます。
- 評価やコメントなどを通して情報検索の手助けとすることができます。
- 情報の共有を通して、コミュニケーションの幅を広げ、推しクリエーターの応援に繋げることができます。

●動画投稿者向けの価値

- 自身の動画を届ける機会に繋げることができます。
- リクエスト通知機能を利用して、動画テーマの参考にすることができます。

- 任意の「編成動画」の情報が一覧で確認できる点。
